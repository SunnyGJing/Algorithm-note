1. 截取文件前n行（前100行）
   ```bash
   >> head -n 100 pre_file_name > target_file_name
   ```
2. 截取文件后n行（后100行）
   ```bash
   >> tail -n 100 pre_file_name > target_file_name
   ```
3. 截取文件中间n行（第100~200行）
   ```bash
   >> sed -n `100, 200p` pre_file_name > target_file_name
   ```
4. 查看文件行数
   ```bash
   >> wc file_name
   ```
